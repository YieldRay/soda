# Fonts

To ensure optimal clarity and usability in various component states, it is highly recommended to  
use a font that supports a medium weight (`font-weight: 500`).  
The `Roboto` font is a good choice for this purpose.

To use `Roboto`, simply incorporate the following lines into your CSS:

```css
/* using Roboto */
@import url('https://fonts.googleapis.com/css2?family=Roboto:wght@100;200;300;400;500;700;900&display=swap');
* {
    font-family: 'Roboto', sans-serif;
}
```
